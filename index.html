Screening Criteria & Weights
🔗 Unified Scoring — Analogue ↔ Screening Bridge ▾
When enabled, basin screening sub-scores are derived from actual analogue reservoir data instead of independent estimates. Reservoir quality is computed from analogue porosity, permeability, and NTG; source rock quality from TOC, Ro, and kerogen type; charge efficiency from migration distance and expulsion efficiency. Screening and analogue search then share a common scoring ontology.
Analogue-Informed Scoring
Screening sub-scores (Source, Reservoir, Seal, Charge, Commercial) incorporate reservoir-level data from the analogue database when analogues exist for a basin.
Analogue Weight in Composite 40%
Blending ratio: how much analogue-derived evidence overrides the basin-level prior. 0% = pure basin estimate, 80% = analogue-dominated.
Source ← Analogue
TOC, Ro vitrinite, Kerogen type, HI, Expulsion efficiency
Reservoir ← Analogue
Porosity, Permeability, NTG, Recovery factor, Depth, API
Charge ← Analogue
Migration distance, Charge volume, Oil family, HC phase
How it works: For each basin, the engine locates matching analogues (by basin name or geological similarity), computes per-element scores from their reservoir properties, then blends those with the basin-level prior using the weight above. Basins with more analogues get higher analogue weight (Bayesian updating).
